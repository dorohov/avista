Программа предмета

Особенности программы
  1. Предкурсовая подготовка:пробные тесты будут оцениваться экспертами по проведнию ЕГЭ по математике
  2. Полная отработка формата ЕГЭ по математике 2017: на каждом занятии отрабатывается определенная тема и экзаменационный навык и один тип заданий ЕГЭ. Пройденный материал каждый раз закрепляется практическим заданием
  3. Регулярное отслеживание знаний ученика по математике: регулярные тесты помогают следить за развитием навыков и знаний ученика по математике
  4. Индивидуальный подход: после каждого теста по математике ученика индивидуально консультирует преподаватель, он показывает сильные и слабые зоны ученика, объясняет тонкости и нюансы, дает ценные рекомендации
  5. По результатам программы ученик ознакомиться со всеми типами заданий ЕГЭ по математике 2017, разберет часто допускаемые ошибки, выработает стратегию выполнения заданий, получит четкое понимание требований к ответам, научится правильно распределять время между разными типами заданий
Строение программы
  1. Диагностика: программа начинается с диагностики знаний по математике, по результатам которой проводится личная беседа с учеником и родителями и определяются цели, основные этапы личной работы каждого ученика. Разрабатывается индивидуальный план обучения
  2. Основной этап обучения: 96 тематических уроков, более 500 заданий, решенных в классе и онлайн, игровые форматы, проверочные и контрольные тесты и работы, личные встречи преподавателя с учеником и родителями
  3. Заключительные упражнения:ученики систематизируют все полученные знания по математике для подготовки к ЕГЭ 2017, готовятся к процедуре сдачи ЕГЭ по математике 2017
Содержание программы «ЕГЭ по математике 2017»

Алгебра:

  • решение уравнений и неравенств: рациональные, иррациональные, тригонометрические, логарифмические, показательно-степенные
  • решение тестовых задач на темы:проценты, производительность и движение, прогрессии, вероятность,реальная математика
  • анализ функций: построение графиков функций и анализ коэффициентов, производная и ее свойства, анализ промежутков монотонности, работа с параметром

Геометрия:

  • планиметрия: основные фигуры и их свойства, вектора и системы координат, основные формулы
  • стереометрия: основные фигуры и их свойства, понятие перпендикуляра, двугранного угла, плоскости, работа с элементами в пространстве
Результаты программы
Программа охватывает все темы, которые проверяет ЕГЭ по математике 2017. Каждая тема и задание ЕГЭ по математике 2017 встретится ученику более 5 раз.
Это позволяет ученику максимально отработать материалы ЕГЭ по математике 2017
